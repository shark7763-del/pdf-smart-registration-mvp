# PDF 智慧解析比賽報名系統 MVP

跆拳道／武術比賽用的「報名表自動化」工具：上傳競賽規程 PDF，系統解析出量級、組別、收費與附件需求，自動生成家長手機報名表，並提供教練後台審核、CSV 匯出與 LINE 通知文字產生。

**單一 HTML 檔、純前端、零安裝**，用瀏覽器打開 `index.html` 即可使用。

🔗 **線上版（GitHub Pages）：** https://shark7763-del.github.io/pdf-smart-registration-mvp/

---

## 功能

分三個畫面（右上「教練功能」切換）：

### 1. 比賽活動報名（家長端 `parentView`）
- 選擇活動 → 填寫學生／家長資料 → 選報名項目與組別
- **量級自動配對**：輸入體重即時推算所屬量級
- **費用自動試算**：依項目／組別套用收費規則
- 附件上傳（照片、級證／段證、切結書、繳費證明…）
- 送出前彈出資料確認，送出後顯示報名編號與繳費方式

### 2. PDF 解析（教練端 `parserView`）
- 用 [pdf.js](https://mozilla.github.io/pdf.js/) 擷取規程 PDF 文字
- 解析結果分頁（基本資料／對練量級／品勢組別／競速踢擊／收費標準／附件需求）可逐欄人工修正
- **解析信心檢查**：低信心或缺漏欄位會提示送出前人工核對
- 確認後「建立報名活動」

### 3. 教練後台（`adminView`，預設密碼 `1234`）
- 報名名單表格：繳費狀態、附件齊全度、審核狀態（待審／通過／退件）
- 統計：總報名／未繳費／附件缺漏
- 收款設定（匯款帳號／LINE Pay／備註）
- LINE 通知產生器（報名成功／未繳費／附件缺漏／裝備／過磅）
- 一鍵匯出 CSV

## 使用方式

1. 線上版直接點上方網址；或本機用瀏覽器開啟 `index.html`。
2. 第一次可按家長端的「載入示範資料」或解析端的範例，體驗完整流程。
3. 進「教練後台」需輸入密碼（預設 `1234`，可在後台修改）。

所有資料目前存在瀏覽器 `localStorage`（本機、單一裝置）。

---

## 已知限制 ⚠️

這是 **MVP**，下列為現階段刻意未處理、需後端才能根治的限制：

1. **資料只在單一瀏覽器**：家長在自己手機填的報名，教練的後台看不到（localStorage 是各裝置私有）。目前僅適合單機自用／展示。
2. **附件容量極小**：附件以 base64 暫存在 localStorage，全域上限約 5MB，數張照片就會塞滿。
3. **「智慧解析」目前是模擬**：`mockAIParse()` 不論上傳什麼 PDF 都回傳示範資料，尚未串接真實 AI。
4. **教練密碼為前端驗證**：明碼存在瀏覽器，任何人開 DevTools 可繞過，僅作展示用途。

## Roadmap

- **Phase 0 — 純前端修補（✅ 已完成 2026-06）**
  報名儲存爆容量不再靜默失敗、全面 XSS 跳脫、繳費改教練手動確認、CSV 公式注入防護、移除死碼與重複事件綁定。通過 jsdom 端到端驗證 11/11。
- **Phase 1 — 後端化**
  接 Google Apps Script + Sheets：報名寫雲端、附件改上 Google Drive、教練改後端驗證。解決限制 1、2、4。
- **Phase 2 — 真 AI 解析**
  以 GAS 當代理呼叫 Claude API，把規程文字轉成結構化 JSON，信心分數改為模型真實回傳。解決限制 3。
